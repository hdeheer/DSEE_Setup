porttest <host> <port>
	A simple perl program to test if the local host can connect to a
	specified TCP port on a specified host. Outputs report to stdout.

_ldif-format_
	"Pretty printer" for unwrapped LDIF data from STDIN.

_ldif-unwrap_
	Unwrapper for LDIF data from STDIN.

fixupldif
	Removed DSEE operational LDAP attributes from LDIF data from STDIN.

ldifformat
	Simple wrapper for _ldif-format_ and _ldif-unwrap_.
